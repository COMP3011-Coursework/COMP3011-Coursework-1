\section{Generative AI Conversation Logs}
\label{app:genai}

This appendix contains exported logs of all Generative AI conversations used during
the development of the Global Food Price Monitor API.
Two sources are represented: \textbf{Claude.ai web conversations} (shared links;
full transcripts accessible via browser) and \textbf{Claude Code sessions} (CLI
agentic sessions; raw logs exported from local session store).
