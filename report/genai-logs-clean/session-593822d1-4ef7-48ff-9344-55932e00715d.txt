## **Human** _2026-03-15 00:00 UTC_

What is lifespan in main.py

---

## **Assistant** _2026-03-15 00:00 UTC_

`lifespan` is a FastAPI feature for running code at application startup and shutdown. It's an async context manager --- everything before `yield` runs on startup, everything after runs on shutdown.

In this project's [backend/app/main.py](backend/app/main.py):

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    async with engine.begin() as conn:
        await conn.run_sync(Base.metadata.create_all)  # <- runs on startup
    await _auto_seed()                                  # <- runs on startup
    yield                                               # <- app runs here, serving requests
                                                        # (shutdown code would go after yield)
```

So every time uvicorn starts the app, it:
1. Creates all database tables (if they don't exist yet)
2. Seeds data from CSV files (if the DB is empty)
3. Then starts serving HTTP requests

It replaced the older `@app.on_event("startup")` decorator, which is now deprecated in FastAPI.

## **Assistant** _2026-03-15 01:18 UTC_

**1. Start only the database:**
```bash
docker compose up db
```

**2. Run backend locally** --- the `DATABASE_URL` uses `db` as the host (Docker service name), which won't resolve outside Docker. Override it for local dev:
```bash
conda activate COMP3011-Coursework-1
cd backend
DATABASE_URL=postgresql+psycopg://foodprice:foodprice123@localhost:5432/foodpricedb uvicorn app.main:app --reload
```

**3. Run frontend locally:**
```bash
cd frontend
npm run dev
```

The Vite proxy already points to `http://backend:8000` --- that also won't resolve locally. You'd need to either:
- Set `VITE_API_URL=http://localhost:8000/api/v1` in a local `.env` file (it's already in your `.env`), or
- Change the proxy target in [frontend/vite.config.ts](frontend/vite.config.ts) to `http://localhost:8000`

The cleanest approach: create a `.env.local` file in `frontend/` (git-ignored) with:
```
VITE_API_URL=http://localhost:8000/api/v1
```
This overrides the Docker value without touching the committed `.env`.
