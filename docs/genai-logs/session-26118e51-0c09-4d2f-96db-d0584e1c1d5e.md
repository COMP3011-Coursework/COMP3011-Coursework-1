## Configure Claude Code to test your MCP server

```bash
claude mcp add --transport http food-price http://localhost:8000/mcp
```

```bash
claude mcp add --transport http food-price --scope project http://localhost:8000/mcp
```
This creates/updates `.mcp.json` in your project root.

**Or add manually to `.mcp.json` in your project root:**
```json
{
  "mcpServers": {
```
